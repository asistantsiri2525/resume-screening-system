# 📄 Resume Screening System

A Flask-based web application that matches resumes with job descriptions using NLP techniques.

---

## 🚀 Features
- Upload multiple resumes (PDF)
- Match resumes with Job Description
- Skill-based filtering
- TF-IDF and cosine similarity scoring
- Top candidate detection 🏆
- Dashboard with analytics 📊

---

## 🛠 Technologies Used
- Python
- Flask
- HTML, CSS
- Scikit-learn
- NLP (TF-IDF, Cosine Similarity)
